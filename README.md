## Project Description

This is a Python GUI-based BMI Calculator developed using Tkinter and SQLite.

The application calculates the Body Mass Index (BMI), displays the BMI category, saves records in a database, and allows users to view their BMI history.

---

## Features

- Calculate BMI
- Display BMI Category
- Save BMI Records
- View BMI History
- Clear Input Fields
- SQLite Database
- User-Friendly GUI

---

## Technologies Used

- Python
- Tkinter
- SQLite
